# Ordenação por Comparação 2

## Objetivo

Implementar e testar:

* Merge Sort
* Quick Sort

---

## Execução

```bash
python main.py data/arquivo.in merge
python main.py data/arquivo.in quick
```

---

## Execuções em lote

```bash
python main.py all
python main.py merge_all
python main.py quick_all
python main.py clear
```

---

## Saída

Arquivo `resultados.txt`:

```
arquivo | algoritmo | tempo
```

---

## Complexidade

| Algoritmo  | Complexidade     |
| ---------- | ---------------- |
| Merge Sort | O(n log n)       |
| Quick Sort | O(n log n) médio |

---

## 🛠️ Requisitos

* Python 3.x
